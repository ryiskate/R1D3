1. Game Overview

PeacefulFarm is a relaxing animal collection and care simulation game where players capture wild animals, raise them on their farm, and build relationships with them. Players use food as bait to capture animals, care for them in specialized habitats, and collect gifts that animals provide once bonded.

Game Pillars

  • Animal Collection: Capture different animal species using appropriate food baits
  • Animal Care: Deep systems for caring for animals and building bonds with them
  • Economy Loop: Animals provide gifts that can be sold for money to buy food and habitats
  • Breeding System: Animals with high happiness and bond can produce eggs that hatch into color variants
  • Farm Development: Gradual expansion by adding new habitats and animal types
  • Atmosphere: Soothing audio and visuals that create a calming experience with day/night cycles

Core Gameplay Loop

  1. Collect food (from trees or by purchasing)
  2. Use food to capture animals in the wild through mini-games
  3. Build and place habitats for captured animals
  4. Feed and care for animals to increase happiness and bond
  5. Collect gifts from bonded animals
  6. Sell gifts to merchants for money
  7. Use money to buy more food and habitats
  8. Breed animals to get color variants

Animal Capture System

The animal capture system works as follows:

  • Food Baits: Each animal type is attracted to a specific food
  • Turtle → Apple
  • Parrot → Corn
  • Jaguar → Meat
  • Rabbit → Carrot
  • Capybara → Lettuce
  • Armadillo → Grubs
  • Tapir → Fruit Mix
  • Capture Areas: Special locations where wild animals can be found
  • Mini-games: Skill-based challenges to successfully capture animals
  • Inventory: Captured animals are added to player inventory

Animal Care and Gifts

The animal care and gift system includes:

  • Regular feeding to maintain and increase happiness
  • After 3 feedings, animals begin to provide gifts
  • Each animal can provide 3 different gifts based on bond level
  • Gifts can be sold to merchants for money
  • Higher bond levels unlock better, more valuable gifts

Breeding System

The breeding system works as follows:

  • Two animals in the same habitat with full bond (level 3) and high happiness (3+)
  • Chance to produce an egg after meeting conditions
  • After parent animals provide 3 more gifts, the egg hatches
  • Offspring have color variations from parents

Animal Happiness System

The animal happiness system has the following components:

  • Happiness scale (0-5)
  • Feeding timer that decreases over time
  • Habitat cleanliness affecting happiness
  • Happiness rules based on feeding timing and habitat cleanliness

Bond Level System

The bond level system works as follows:

  • Bond levels range from 0-3
  • Bond increases when happiness reaches certain thresholds
  • Higher bond levels require more happiness points
  • Bond decreases if happiness would go below -1
  • Higher bond levels unlock better gifts

Gift System

The gift system works as follows:

  • After 3 feedings, animals begin to provide gifts
  • Each animal type can provide 3 different gifts based on bond level
  • Bond Level 1: Basic gift
  • Bond Level 2: Improved gift
  • Bond Level 3: Rare gift
  • Gifts can be sold to merchants for money

Economy Loop

The game's economy follows a sustainable loop:

  1. Start with a turtle habitat and apple tree
  2. Collect apples to capture turtles
  3. Feed turtles to increase happiness and bond
  4. Collect gifts from bonded turtles
  5. Sell gifts to merchants for money
  6. Use money to buy food for existing animals
  7. Use money to buy new habitats and food for capturing new animals
  8. Expand collection with new animal types
  9. Breed animals to get color variants

Merchant System

The merchant provides the following services:

  • Buying animal gifts at different prices based on rarity
  • Selling food items needed for animal care and capture
  • Selling new habitat types as player progresses
  • Potentially buying animals from the player

Gift Values and Economy Progression

The economy is designed to encourage players to invest in raising animals' bond levels:

  • Bond Level 0: No gifts yet, player must invest in feeding
  • Bond Level 1: Gifts roughly equal feeding costs (break-even)
  • Bond Level 2: Gifts exceed feeding costs (modest profit)
  • Bond Level 3: Gifts significantly exceed feeding costs (substantial profit)

Gift System

The gift system uses physical objects that appear in the world:

  • Gift Generation: Animals produce gifts after being fed a certain number of times (default: 3 feeds)
  • Physical Representation: Gifts appear as interactive objects in the animal's habitat
  • Animations: Gifts bob up and down and rotate slowly to attract attention
  • Interaction: Players must click on gifts to collect them
  • Value Decay: Gift value decreases over time (5% per hour, up to 70% maximum)
  • Persistence: Gifts remain in the world across scene changes and game sessions
  • Visual Indicators: UI elements show when gifts are ready and when they're collected
  • Gift Catalog: Players can view all discovered and undiscovered gifts in a catalog UI

Gift Catalog UI

The Gift Catalog provides players with information about all gifts in the game:

  • Discovery System: Gifts are initially shown as silhouettes until collected for the first time
  • Filtering: Players can filter gifts by animal type or bond level
  • Gift Details: Each entry shows the gift's name, description, value range, and source animal
  • Collection Tracking: The catalog tracks how many of each gift type the player has collected
  • Completion Rewards: Bonuses are awarded for completing animal gift sets

Habitat Progression

The habitat system follows a progression path:

  • Player starts with a turtle habitat and apple tree
  • As players earn money from selling gifts, they can purchase new habitats
  • Each habitat type houses a specific animal species
  • Players can choose where to place new habitats on their farm
  • Existing habitats can be removed and replaced with different types

Habitat Cleanliness System

The habitat cleanliness system includes:

  • Dirt objects that appear over time
  • Cleaning interaction to remove dirt
  • Dirt count affecting animal happiness
  • Dirt respawn timer

Habitat Capacity

Each habitat has specific characteristics:

  • Can house multiple animals of the same type
  • Animals in the same habitat can breed if conditions are met
  • Cleanliness affects all animals in the habitat

Habitat Management

Players can manage their habitats by:

  • Purchasing new habitats from the merchant
  • Selecting a free spot on the farm to place habitats
  • Removing existing habitats to make room for new ones
  • Maintaining cleanliness to keep animals happy

11. Unity Configuration Guide

This section provides detailed instructions for configuring the various systems in the Unity Editor for PeacefulFarm.

Gift System Configuration

  1. Create Gift Prefab:
    • Create an empty GameObject in your scene
    • Add components: SpriteRenderer, Box Collider 2D, and GiftObject script
    • Add a child GameObject named "SelectionIndicator" with a SpriteRenderer
    • Save as a prefab in your Prefabs folder
  2. Configure GiftManager:
    • Create a GiftManager GameObject in your scene
    • Assign the Gift Prefab reference
    • Set Gift Decay Rate Per Hour to 0.05 (5% per hour)
    • Set Max Decay Percent to 0.7 (70% maximum)
    • Assign references to PlayerAnimalInventory, PlayerGiftInventory, and MarketSystem
  3. Create Gift Data:
    • Right-click in Project window > Create > PeacefulFarm > Economy > Gift
    • Configure each gift with name, animal type, bond level, value range, and sprites
    • Add all created Gift Data ScriptableObjects to the GiftManager's "All Gifts" array
  4. Configure AnimalGiftProvider:
    • Ensure each animal prefab has the AnimalGiftProvider component
    • Set Feeds Required For Gift to 3 (or preferred number)
    • Assign references to GiftManager and PlayerAnimalInventory
  5. Set Up GiftSaveSystem:
    • Create a GiftSaveSystem GameObject in your scene
    • Set Save File Name to "gifts.json"
    • Assign the GiftManager reference

Habitat System Configuration

  1. Habitat Prefab Setup:
    • Create a base habitat prefab with required components: HabitatManager, HabitatDirtController
    • Add 9 dirt GameObjects as children (can be named anything)
    • Create variants for each animal type (Turtle, Jaguar, etc.) and save in Resources/Prefabs/Habitats
  2. Configure HabitatManager:
    • Assign habitat prefab references for each animal type
    • Set up habitat slot transforms in the scene
    • Configure habitat costs in the inspector
  3. Set Up HabitatSaveSystem:
    • Create a HabitatSaveSystem GameObject
    • Set Save File Name to "habitats.json"
    • Assign the HabitatManager reference

Animal System Configuration

  1. Animal Prefab Setup:
    • Create a base animal prefab with required components: AnimalIdentifier, AnimalHappinessManager, AnimalGiftProvider, NavMeshAgent
    • Configure NavMeshAgent for proper movement (speed, stopping distance, etc.)
    • Create variants for each animal type and save in Resources/Prefabs/Animals
  2. Configure AnimalHappinessManager:
    • Set Time To Eat parameters (min/max hours)
    • Configure happiness thresholds and bond level requirements
  3. Set Up AnimalSaveSystem:
    • Create an AnimalSaveSystem GameObject
    • Set Save File Name to "animals.json"
    • Assign the AnimalManager reference

Audio System Configuration

  1. Audio Mixer Setup:
    • Create an Audio Mixer (Window > Audio > Audio Mixer)
    • Create groups: Master, Music, SFX, and Ambience
    • Set up volume parameters for each group
  2. Configure AudioManager:
    • Create an AudioManager GameObject with AudioManager script
    • Assign the Audio Mixer reference
    • Create and assign SoundCollection ScriptableObjects for different sound categories
  3. Set Up AmbienceManager:
    • Create an AmbienceManager GameObject
    • Assign day/night music clips
    • Configure transition times and fade durations

UI System Configuration

  1. Canvas Setup:
    • Create a main Canvas with CanvasScaler (set to Scale With Screen Size)
    • Create separate canvases for different UI layers (HUD, Menus, Notifications)
  2. Configure GiftNotificationUI:
    • Create a notification panel prefab with Text and Image components
    • Add the GiftNotificationUI script
    • Configure animation parameters (duration, scale curve)
  3. Set Up MarketUI:
    • Create market panel prefabs with necessary UI elements
    • Configure item slots and button callbacks
    • Assign references to MarketSystem and PlayerInventory

Scene Setup

  1. Main Farm Scene:
    • Place habitat slots in a circular arrangement
    • Add NavMesh surface and bake for animal movement
    • Place food sources (apple trees, etc.)
    • Add all manager GameObjects (GiftManager, HabitatManager, etc.)
  2. Persistent Managers:
    • Create a PersistentManagers prefab with DontDestroyOnLoad
    • Include SaveManager, AudioManager, and other cross-scene managers
    • Use a bootstrap scene to initialize these managers

Testing Configuration

  1. Debug Mode:
    • Enable debug flags on key managers for detailed logging
    • Create test buttons for spawning animals, gifts, etc.
    • Use the Debug build configuration for development testing
  2. Save Data Management:
    • Use PlayerPrefs to store configuration options
    • Create a debug menu for clearing save data
    • Implement save data versioning for compatibility
