R1D3 Project Pages Map

Overview

This document maps the template structure and relationships for the projects section of the R1D3 application.

Template Hierarchy

Base Templates

  • base.html (root template)
    • core/base_task_dashboard.html (specialized dashboard base)
    • projects/unified_task_form.html (base for all task forms)

Task Management Templates

General Task Templates

  • task_dashboard.html (extends core/base_task_dashboard.html)
  • task_list.html (extends base.html)
  • task_detail.html (extends base.html)
  • task_form.html (extends base.html)
  • task_kanban.html (extends base.html)
  • task_confirm_delete.html (extends base.html)

Specialized Task Templates

URL Structure and Page Flow

Main Project Pages

  • Dashboard: / → ProjectDashboardView → projects/dashboard.html
  • Project List: /list/ → ProjectListView → projects/project_list.html
  • Project Detail: // → ProjectDetailView → projects/project_detail.html
  • Project Create: /create/ → ProjectCreateView → projects/project_form.html
  • Project Update: //update/ → ProjectUpdateView → projects/project_form.html

Task Management

  • Task List: /tasks/ → TaskListView → projects/task_dashboard.html
  • Task Detail: /tasks// → TaskDetailView → projects/task_detail.html
  • Task Create: /tasks/create/ → TaskCreateView → projects/task_form.html
  • Task Update: /tasks//update/ → TaskUpdateView → projects/task_form.html
  • Task Delete: /tasks//delete/ → TaskDeleteView → projects/task_confirm_delete.html
  • Task Kanban: /tasks/kanban/ → TaskKanbanView → projects/task_kanban.html

Game Management

  • Game Dashboard: /games/ → GameDashboardView → projects/game_dashboard.html
  • Game List: /games/list/ → GameProjectListView → projects/game_list.html
  • Game Detail: /games// → GameProjectDetailView → projects/game_detail.html
  • Game Create: /games/create/ → GameProjectCreateView → projects/game_form.html
  • Game Update: /games//update/ → GameProjectUpdateView → projects/game_form.html
